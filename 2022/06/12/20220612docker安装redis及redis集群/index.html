docker安装redis及redis集群

需要自己准备一个redis.conf文件

redis中文官方网站：http://www.redis.cn/download.html 下载解压然后获取文件

docker版本20.10.14

单机模式

  • 获取docker镜像

    docker pull redis

  • 修改准备好的redis.conf文件

  • 把上面的文件放入某个目录

我这里放入 /usr/redis/redis.conf

  • 启动

集群模式

启动多个redis（可以在多个服务器上或者在一个上）

创建并运行docker容器实例

–name redis-node-6 容器名字
–net host 使用宿主机的IP和端口，默认
–privileged=true 获取宿主机root用户权限
-v /data/redis/share/redis-node-6:/data 容器卷，宿主机地址:docker内部地址
redis:6.0.8 redis镜像和版本号
–cluster-enabled yes 开启redis集群
–appendonly yes 开启持久化
–port 6386 redis端口号

构建集群关系与测试

  • 随机选取一个节点作为主节点,并进入容器
  • 构建集群
  • 查看集群信息

主从容错切换分析与实现

  • 当前的三主三从
  • 停止节点1
  • 过一分钟左右，进入节点2查看集群节点，可以看到1对应的5直接成为了master
  • 重新启动节点1，可以看到节点5任然是master，节点1变成了slave
  • 如果你还行让节点1为主机，节点5成从机，那么就先停掉，节点5，在启动节点5….
